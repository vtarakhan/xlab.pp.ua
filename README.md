# xlab.pp.ua

Простий статичний сайт з ефектом «Matrix» і анімацією завантаження. Проєкт призначений для демонстрації мінімалістичного візуального оформлення та може використовуватися як базова сторінка для персонального домену.

## Опис проекту

Сайт складається з однієї сторінки з:

- чорним фоном;
- анімацією Matrix-подібного потоку символів;
- текстом «SYSTEM INITIALIZING...» з плавним ефектом завантаження;
- адаптивною версткою для мобільних пристроїв і десктопів.

## Технології

- HTML5
- CSS3
- JavaScript (vanilla)
- Статичний хостинг через домен xlab.pp.ua

## Структура файлів

- `index.html` — головна сторінка
- `static/main.css` — стилі
- `static/matrix.js` — анімація Matrix
- `static/loading.js` — ефект тексту завантаження
- `CNAME` — налаштування кастомного домену

## Як запустити локально

1. Відкрийте термінал у корені проєкту.
2. Запустіть локальний сервер, наприклад:

   ```bash
   python -m http.server 8000
   ```

3. Відкрийте у браузері:

   ```text
   http://localhost:8000
   ```

## Як розгорнути

Проєкт є статичним, тому його можна розгорнути на будь-якому хостингу статичних файлів, наприклад:

- GitHub Pages
- Netlify
- Vercel
- власний веб-сервер

Для кастомного домену використовується файл `CNAME` з записом:

```text
xlab.pp.ua
```

## Примітка

Це невеликий демонстраційний проєкт із мінімальним набором файлів, який легко розширювати або використовувати як основу для більш складного сайту.
